Detailed Description

template<class K, class V, uint8_t L = sizeof(V) * 8, uint8_t DL = 0>
class DataPlaneOthello< K, V, L, DL >

Describes the data structure l-Othello. It classifies keys of keyType into 2^L classes. The array are all stored in an array of uint64_t. There are actually m_a+m_b cells in this array, each of length L.

Note
Be VERY careful!!!! valueType must be some kind of int with no more than 8 bytes' length
