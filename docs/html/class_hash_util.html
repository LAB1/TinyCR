static uint32_t HashUtil::BobHash32 ( const void *  key,
size_t  key_size 
)
inlinestatic

Every bit of the key affects every bit of the return value. Every 1-bit and 2-bit delta achieves avalanche. About 6 * length + 35 instructions.

Use for hash table lookup, or anything where one collision in 2^32 is acceptable. Do NOT use for cryptographic purposes.
